WhatsApp CRM and customer communication solution for WooCommerce powered by the Meta WhatsApp Cloud API.

== Description ==

Ask I Viki CRM for WooCommerce transforms customer communication into a complete messaging and CRM experience powered by the Meta WhatsApp Cloud API.

Manage customer conversations, monitor order activity, send WhatsApp notifications, organize customer profiles, and provide faster support directly from your WordPress dashboard.

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory.
2. Activate the plugin through the WordPress Plugins screen.
3. Navigate to WooCommerce → Ask I Viki CRM.
4. Configure your Meta WhatsApp Cloud API credentials.
5. Configure the webhook URL in Meta Developer Portal.
6. Save settings and test your connection.
7. Start managing customer conversations from the WhatsApp CRM Inbox.

== Frequently Asked Questions ==

= Does this plugin require WooCommerce? =

Yes. WooCommerce must be installed and activated.

= Does this plugin use the official WhatsApp API? =

Yes. The plugin integrates with the official Meta WhatsApp Cloud API.

= Can I send order notifications automatically? =

Yes. Processing, completed, and cancelled order notifications can be sent automatically.

= Can I manage customer conversations inside WordPress? =

Yes. The built-in CRM inbox allows you to view and manage customer conversations directly from the WordPress admin dashboard.

= Does the plugin support quick replies? =

Yes. You can create reusable response templates for faster customer support.
